# DevHaven v2.8.0 Release Notes

- 发布日期：2026-03-05
- Tag：`v2.8.0`
- 对比版本：`v2.7.2` -> `v2.8.0`

## 本次升级内容

### 1. 新增 Web 运行时（浏览器可用）
- 引入浏览器运行时桥接层，前端命令与事件通道在 Tauri / Web 间自动切换。
- 新增本地 Rust Web 服务（HTTP + WebSocket），支持 `/api/cmd/*` 命令分发与实时事件推送。
- 开发态支持单端口体验与 `/api` 代理，浏览器访问本地服务更直接。

### 2. 项目管理稳定性修复
- 修复“直接添加项目”在刷新后丢失的问题，保证手动添加的项目路径可持续保留。

### 3. 终端显示细节优化
- 终端字体栈优先使用 Nerd Font，图标/符号渲染更完整。

## 版本升级

本次发布将应用版本统一升级为 `2.8.0`：
- `package.json`
- `package-lock.json`
- `src-tauri/tauri.conf.json`
- `src-tauri/Cargo.toml`
- `src-tauri/Cargo.lock`

## 兼容性说明

- 无新增迁移步骤。
- 本地数据目录仍为 `~/.devhaven/`。
