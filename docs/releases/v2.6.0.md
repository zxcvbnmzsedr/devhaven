# DevHaven v2.6.0 Release Notes

- 发布日期：2026-02-26
- Tag：`v2.6.0`
- 对比版本：`v2.5.2` -> `v2.6.0`

## 本次升级内容

### 1. 全局 Skills 管理能力上线
- 新增全局 Skills 管理弹窗，可统一查看与管理已安装/可安装技能。
- 支持内置全局技能一键安装，减少手工配置成本。
- 支持按矩阵视图浏览技能，提升可发现性和批量管理效率。

### 2. Skills 矩阵交互优化
- 增加矩阵视图安装/卸载切换操作，流程更顺滑。
- 优化长文本截断与 Tooltip 展示，避免信息被遮挡。
- 前后端补齐技能状态模型与命令链路，减少状态不一致风险。

### 3. 工程接线补齐
- 前端新增 `skills` 服务层并在主界面接入入口。
- Tauri 后端新增 Skills 相关命令与实现，形成完整闭环。
- `AGENTS.md` 已同步回写 Skills 功能地图，便于后续定位与维护。

## 版本升级

本次发布将应用版本统一升级为 `2.6.0`：
- `package.json`
- `package-lock.json`
- `src-tauri/tauri.conf.json`
- `src-tauri/Cargo.toml`
- `src-tauri/Cargo.lock`

## 兼容性说明

- 无新增迁移步骤。
- 本地数据目录仍为 `~/.devhaven/`，新增字段保持兼容处理。
