# DevHaven v2.5.2 Release Notes

- 发布日期：2026-02-26
- Tag：`v2.5.2`
- 对比版本：`v2.5.1` -> `v2.5.2`

## 本次升级内容

### 1. 项目列表效率升级
- 支持项目收藏并置顶展示，常用项目更容易命中。
- 列表卡片/行内支持直接运行项目脚本，减少进入详情页的往返。
- 新增多选批量操作（复制路径、刷新、打标、移入回收站）。

### 2. 全局命令面板
- 新增 `⌘K / Ctrl+K` 全局命令面板。
- 支持项目跳转、脚本运行、筛选切换等高频操作。

### 3. 信息可见性增强
- 热力图支持按日期下钻查看当天活跃项目并快速定位。
- 项目列表新增最近提交摘要（Git 项目展示提交信息，非 Git 项目显示占位）。
- 项目详情新增 `PROJECT_TODO.md` 读写能力，支持勾选、增删与自动保存。

## 版本升级

本次发布将应用版本统一升级为 `2.5.2`：
- `package.json`
- `package-lock.json`
- `src-tauri/tauri.conf.json`
- `src-tauri/Cargo.toml`
- `src-tauri/Cargo.lock`

## 兼容性说明

- 无新增迁移步骤。
- 本地数据目录仍为 `~/.devhaven/`，新增字段均做向后兼容处理。
