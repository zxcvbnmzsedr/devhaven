# DevHaven v2.7.1 Release Notes

- 发布日期：2026-02-28
- Tag：`v2.7.1`
- 对比版本：`v2.7.0` -> `v2.7.1`

## 本次升级内容

### 1. 应用主流程与终端模块重构
- `App.tsx` 按领域拆分为多个 hooks（视图状态、筛选、选择、终端、命令面板、Codex 集成等），降低单文件复杂度。
- 终端快捷命令链路拆分为运行时、派发层与面板层，交互逻辑和状态机职责更加清晰。

### 2. 前后端性能优化
- 前端：主列表增加虚拟滚动，并通过 `memo/useCallback` 降低高频重渲染成本。
- 后端：项目扫描引入 Rayon 并行化，提升大目录下的扫描效率。
- 后端：优化 Codex 监控、Git Daily 统计、备注预览读取与终端 UTF-8 处理，减少卡顿与异常字符问题。

### 3. 稳定性修复
- 修复应用状态更新链路中的竞态与覆盖问题，提升状态一致性。
- 修复终端工作区持久化与恢复的边界问题，减少会话状态丢失。

### 4. 发布质量门禁增强
- 发布工作流新增前端构建与 Rust `cargo check --locked` 质量门禁，降低带病发布风险。
